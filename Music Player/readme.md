Music Player Project

Overview:
This is a web-based music player built using HTML, CSS, and JavaScript. It allows users to play, pause, and control the volume of songs, as well as manage a playlist.

Features:
- Play/Pause functionality
- Volume control
- Progress bar for seeking
- Playlist management
- Responsive design

How to Use:
1. Open the `index.html` file in your browser.
2. Click on a song in the playlist to play it.
3. Use the play/pause button, progress bar, and volume slider to control playback.

Technologies Used:
- HTML
- CSS
- JavaScript
